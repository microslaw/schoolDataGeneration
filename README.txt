code can be also downloaded from https://github.com/microslaw/schoolDataGeneration

main.ipynb file is the recomended way of using this generator.
main.py has the exact same code as main.ipynb but in a .py file.

parameters can be modified by editing the parameters.py file.

Setup:
1. Enter into command line:
pip install -r requirements.txt

2a. (recomended)
Download visual studio code and install the jupyter extension
(https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)

or 2b. Enter into command line:
jupyter lab .

or 2c. Enter into command line:
./main.py
